# MANIIKABINET

Многостраничный образовательный web-проект о базовом курсе маникюра, готовый для GitHub Pages.

## Файлы

- `index.html` — главная страница проекта.
- `course.html` — описание курса, аудитории и ценности.
- `route.html` — маршрут обучения и логика прогрессии.
- `modules.html` — программа модулей и методика.
- `practice.html` — задания, упражнения и практика.
- `testing.html` — контрольное тестирование и метрика прогресса.
- `styles.css` — общая дизайн-система, навигация и адаптив.
- `script.js` — mobile nav, scroll progress, хранение прогресса и логика теста.
- `.nojekyll` — отключение Jekyll для GitHub Pages.

## Как открыть локально

Откройте `index.html` в браузере или запустите простой сервер:

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000`.

## Публикация на GitHub Pages

1. Загрузите все файлы в корень репозитория.
2. В GitHub откройте `Settings` → `Pages`.
3. Выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/ (root)`.
5. Сохраните настройки и дождитесь публикации.
